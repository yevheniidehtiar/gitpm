**Phase 1 — CLI**

Port `packages/cli/src/commands/` (Phase 1 commands) to `go/cmd/gitpm/` using cobra.

Commands to port:
- `gitpm init` — scaffold `.meta/` directory
- `gitpm validate` — parse → resolve → validate, display errors/warnings
- `gitpm query` — filter entities with type/status/priority/label flags
- `gitpm show <path>` — display single entity details
- `gitpm create story|epic|milestone` — create entities with flags
- `gitpm set <path> field=value` — mutate entity fields
- `gitpm move <story> --to-epic <epic>` — move story between epics

Terminal UX:
- Use `lipgloss` for styled output (colors, tables)
- Use `bubbles` for spinners/progress on long operations
- `--json` flag on all commands for machine-readable output

**Source reference**: `packages/cli/src/commands/` (~1,200 LOC for these commands)

**Acceptance criteria**:
- All 7 commands produce identical results to TypeScript CLI
- `--json` flag works on every command
- `--help` auto-generated by cobra
